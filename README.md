# Compteur de mots en Java

Ce programme lit un fichier texte, compte les mots ligne par ligne en utilisant l'espace comme délimiteur, et affiche le total.

## Fonctionnement
1. Utilise `java.util.Scanner` pour la lecture du fichier.
2. Sépare chaque ligne avec `split(" ")`.
3. Calcule la longueur du tableau résultant.

## Comment l'exécuter
1. Assurez-vous d'avoir un fichier `monTexte.txt` à la racine du projet.
2. Lancez `CompteurMots.java`.
